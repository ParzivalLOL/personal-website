Body Wearable System for Parkinson's Disease - Project

At Plaksha University, I worked with undergrads in CS/AI to develop a non-invasive method of detecting Parkinson’s Disease in its early stages. As it was a data collection project, we understood that the more sensors we used the better our research would be. Hence, we connected 12 sensors to our subject, spread throughout the body as shown in the photo. I also programmed a system for AI diagnosis, which had an accuracy of 95.2%, which was pretty high considering the limitations of the datasets we could find.

Body Wearable System for Parkinson’s Disease - Sensor

Most sensor boxes include a power supply, a voltage adapter, ESP32 (microcontroller) and an MPU6050 ( accelerometer and gyroscope). Combined, these track the movement of the hand and send the data to a computer using MQTT. This data is stored in a CSV and then plotted using Python. Some sensor boxes also include an EMG sensor (Electromyography). Signals in our body are transferred using electric signals, so the EMG helps to detect muscle activity even if there isn’t a noticeable tremor (as in early stages).

Remote Controlled Car

It is a remote-controlled car which can be operated via smartphones. It uses an ESP32 board for bluetooth communication, 2 DC motors (and the l298n motor driver) which are supplied 9V.
